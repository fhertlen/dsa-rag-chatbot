# DSA RAG Chatbot

Ein KI-Chatbot für Das Schwarze Auge (DSA), der Regelfragen beantwortet, 
Aventurien-Lore erklärt und bei der Abenteuerplanung hilft – basierend auf 
einer persönlichen DSA-Sammlung aus Regelwerken, Quellenbänden und Abenteuern.

## Was kann er?

- Regelfragen beantworten (DSA5)
- Aventurien-Lore erklären (DSA4 & DSA5)
- Bei Abenteuerplanung helfen: NSCs, Szenen, Plot-Ideen
- Quellenangabe: Der Bot nennt immer aus welchem Buch die Antwort stammt

## Datenbasis

- DSA5 Regelwerk
- DSA4 & DSA5 Quellenbände und Abenteuer
- Aventurica Wiki

## Technologie

- RAG (Retrieval-Augmented Generation)
- ChromaDB (Vektordatenbank)
- Claude API (LLM)
- Streamlit (Frontend)
